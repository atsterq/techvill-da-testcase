# techvill-da-testcase

## Задача 1

Нужно автоматизировать поздравление сотрудников с днём рождения: учитывать подразделения и поздравлять **только тех, кто работает сегодня** (по графику дежурств), отправляя уведомление в тестовый канал.

### Источник данных

В качестве тестового хранилища используется SQLite. 

Данные:
- сотрудники: `employees (id, ext_id, dept_id, fio, dob)`
- графики дежурств: `duty_schedules (employee_id, day_of_week, is_working, shift_start, shift_end)`
- подразделения: `departments (id, name)`

### Логика

1. При первом запуске генерируется `EMPLOYEE_COUNT` сотрудников и их график на неделю.
2. Дальше SQL-запросом выбираются сотрудники, которые:
   - работают сегодня (по `duty_schedules` на `weekday(today)`),
   - и у которых сегодня день рождения (сравнение `dob` по `MM-DD`),
   - и подтягивается название отдела.
3. Для каждого найденного сотрудника формируется текст поздравления и отправляется в тестовый канал.

### Запуск

```bash
python birthday_notifier.py
```
